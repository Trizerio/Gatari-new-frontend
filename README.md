# Gatari-new-frontend
- Finally a new frontend coded with Vuejs
## How to set.up 

Install the required dependencies
```
$ npm install
```
And finally start server
```
$ npm run dev
```

And the site will be here: http://localhost:8080/public/


And that's it.
